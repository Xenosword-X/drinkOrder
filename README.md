📚 學習心得
這個專案是我首次使用 Vue.js 進行功能性開發，從資料綁定、事件處理，到條件渲染與即時畫面更新，讓我體驗到前端框架的開發效率與維護性。加深了我對「資料驅動畫面」的理解，也學會如何將 UI 與邏輯清楚地分離，建立有結構的前端應用邏輯。
